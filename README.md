    더 <b>자율적이고 안전한 여행 경험</b>을 제공하는 채팅 기반 매칭 플랫폼
  </p>

  <hr style="width: 70%;"/>
</div>

<br/>

<h2>📌 프로젝트 개요</h2>
<p>
  <b>LocalMate</b>는 여행자와 현지 거주 일반인을 연결하는 <b>현지인 가이드 매칭 서비스</b>입니다.<br/>
  상업적인 투어 중심 서비스가 아닌, <b>실제 현지인의 시선</b>으로 여행을 경험하도록 돕습니다.
</p>

<hr/>

<h2>📌 기획 의도</h2>
<ul>
  <li>요즘 워라벨을 중시하며 <b>가까운 해외로 혼자 여행</b>을 떠나는 사람들이 증가</li>
  <li>하지만 <b>현지 정보 부족</b>, <b>언어 장벽</b>, <b>치안 문제</b>로 불안을 느끼는 경우가 많음</li>
  <li>기존 여행 서비스는 <b>상업적 가이드</b> / <b>정형화된 일정</b> 중심이라 생생한 현지 경험에 한계</li>
</ul>
<p>
  따라서 LocalMate는 <b>현지 거주 일반인과의 직접 소통</b>을 통해<br/>
  여행자가 <b>자율적이고 안전한</b> 여행을 할 수 있도록 기획했습니다.
</p>

<hr/>

<h2>📌 프로젝트 소개</h2>

<div style="border: 1px solid #ddd; border-radius: 12px; padding: 14px;">
  <ul>
    <li><b>채팅 플랫폼</b>을 통해 여행 목적에 맞는 일정/안내를 <b>자율적으로 조율</b></li>
    <li>현지 가이드를 <b>검색</b>하거나, <b>가이드 모드</b>로 직접 활동 가능</li>
    <li>프로필 설정 + 검색 필터로 <b>선호 스타일의 상대</b>를 찾을 수 있음</li>
    <li>신고/제재/공지/통계 등 <b>권한 기반 관리자 기능</b> 제공</li>
  </ul>
</div>

<h2>📌 주요 기능</h2>

<h3>가이드 조회</h3>
<ul>
  <li>가이드 검색 및 프로필 확인</li>
</ul>

<h3>검색 필터링</h3>

  <li>프로필/언어/성향 기반 필터로 원하는 상대 탐색</li>
