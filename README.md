<div align="center">
  <h1>LocalMate</h1>
  <p>
    혼자 떠나는 해외여행, <b>현지인과 직접 연결</b>되어<br/>
    더 <b>자율적이고 안전한 여행 경험</b>을 제공하는 채팅 기반 매칭 플랫폼
  </p>

  <hr style="width: 70%;"/>
</div>

<br/>

<h2>📌 프로젝트 개요</h2>
<p>
  <b>LocalMate</b>는 여행자와 현지 거주 일반인을 연결하는 <b>현지인 가이드 매칭 서비스</b>입니다.<br/>
  상업적인 투어 중심 서비스가 아닌, <b>실제 현지인의 시선</b>으로 여행을 경험하도록 돕습니다.
</p>

<hr/>

<h2>📌 기획 의도</h2>
<p>
  개인의 워라벨이 중요시되는 요즈음, 가까운 해외로 혼자 여행을 떠나는 사람들이 증가하고 있습니다.<br/>
  그러나 <b>현지 정보 부족, 언어 장벽, 치안 문제 등</b>으로 인해 불안을 느끼는 경우가 많습니다.<br/>
  기존 여행 서비스는 상업적인 가이드나 정형화된 일정에 초점을 맞추고 있어,<br/>
  실제 현지의 생생한 여행을 즐기기에는 한계가 있다고 판단했습니다.<br/>
  본 프로젝트는 여행자가 현지에 거주하는 일반인과의 직접적인 소통을 통해 <br/>
  <b>자율적이고, 안전한 여행 경험</b>을 제공하고자 기획하였습니다.
</p>

<hr/>

<h2>📌 프로젝트 소개</h2>

<div style="border: 1px solid #ddd; border-radius: 12px; padding: 14px;">
  <ul>
    <li><b>채팅 플랫폼</b>을 통해 여행 목적에 맞는 일정/안내를 <b>자율적으로 조율</b></li>
    <li>현지 가이드를 <b>검색</b>하거나, <b>가이드 모드</b>로 직접 활동 가능</li>
    <li>프로필 설정 + 검색 필터로 <b>선호 스타일의 상대</b>를 찾을 수 있음</li>
    <li>신고/제재/공지/통계 등 <b>권한 기반 관리자 기능</b> 제공</li>
  </ul>
</div>

<h3>가이드 조회</h3>
<ul>
  <li>가이드 모드 설정된 사용자 조회</li>
  <li>인기순, 접속 시간순 정렬</li>

<h3>검색 필터링</h3>
